Basic

Combo box is created with the following one liner:

    $("#tb1").combobox({ options: ["apple","orange","banana","mangoesteen"] });

Define number of visible items

        $("#tb2").combobox({ options: ["January", "February", "March", "April", "May", "June", 
                                    "July", "August", "September", "October", "November", "December"],
                            size: 5 });
